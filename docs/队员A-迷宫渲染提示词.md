## 粘贴给 Agent 的完整提示词：

```
帮我创建一个迷宫游戏的前端页面，放在 frontend/maze-game/ 目录下。

## 要求：

### 文件结构
在 frontend/maze-game/ 下创建以下文件：
- index.html（页面结构）
- style.css（样式）
- maze.js（迷宫渲染 + 操控逻辑）

### 功能需求：
1. 用 Canvas 画布渲染迷宫，画布大小 500x500，迷宫格子 21x21
2. 调用后端 API 获取迷宫数据：GET http://localhost:5001/api/maze/generate
   后端返回格式：{ "maze": [[0,0,1,0,...], [1,0,0,0,...], ...] }
   其中 0=路, 1=墙
3. 玩家用键盘方向键 ↑↓←→ 控制移动，每次移动一个格子
4. 玩家用绿色圆形表示，起始点在 (1,1)，终点在右下角，用红色标记
5. 到达终点后弹出提示"恭喜通关！"，然后调用后端提交成绩：
   POST http://localhost:5002/api/score
   body: { "player": "玩家名", "time": 通关秒数, "level": 1 }
6. 页面上方显示计时器（从开始移动到通关的时间）
7. 有"重新生成迷宫"按钮，点击后重新获取迷宫
8. 起点固定在左上角(1,1)，终点固定在右下角

### 样式要求：
- 页面居中，深色背景
- 迷宫墙壁用深灰色，道路用白色
- 界面简洁美观

### 用到的数据结构（在注释里标注）：
- 二维数组：存储迷宫地图
- 在 maze.js 里用注释标注 "/* 数据结构：二维数组/矩阵，存储迷宫每个格子的值 */"

请一次性生成所有完整代码，不要省略。
```
