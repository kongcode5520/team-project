### 核心要求：用 DFS（深度优先搜索）+ 栈 生成迷宫

迷宫生成算法必须使用栈数据结构：
1. 初始化一个全为墙(1)的二维数组，大小为 width x height
2. 从 (1,1) 开始，设为路(0)
3. 使用栈来记录访问路径：
   - 将当前格子压入栈
   - 随机选择一个未访问的相邻格子（间隔2格）
   - 打通中间的墙
   - 将该相邻格子设为当前格子，压入栈
   - 如果没有未访问的相邻格子，从栈中弹出（回溯）
   - 重复直到栈为空，迷宫生成完毕
4. 起点永远是 (1,1)，终点永远是 (width-2, height-2)
5. width 和 height 必须是奇数；如果传入偶数，自动 +1 变成奇数
6. 在代码中用注释标注：
   "# 数据结构：栈(Stack)，用于DFS迷宫生成的回溯"
   "# 数据结构：二维数组/矩阵，存储迷宫地图"
   "# 算法：深度优先搜索(DFS) + 回溯"

### 迷宫生成算法图解说明（供理解，注释里要写）：
栈的作用：记录"走过的路"
- 前进时 push：记录当前路径
- 无路可走时 pop：回到上一个分叉点
- 这就是 DFS 的回溯机制
