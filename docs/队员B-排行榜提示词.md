帮我创建一个排行榜前端页面，放在 frontend/leaderboard/ 目录下。

## 要求：

### 文件结构
在 frontend/leaderboard/ 下创建以下文件：
- index.html（页面结构）
- style.css（样式）
- leaderboard.js（数据获取 + 排序展示逻辑）

### ⚠️ 接口规范（必须严格按此格式调用，否则后端收不到！）

你负责调用后端接口，不需要实现后端。以下是你需要调用的接口：

**接口1：获取排行榜 [队员D 实现]**
请求：GET http://localhost:5002/api/leaderboard?level=1

响应格式（必须按此格式解析）：
{
  "scores": [
    {"rank": 1, "player": "张三", "time": 35, "level": 1},
    {"rank": 2, "player": "李四", "time": 42, "level": 1}
  ]
}

字段说明：
- scores: 数组，已按 time 升序排列
- rank: 排名
- player: 玩家名称
- time: 通关时间（秒），越小越好
- level: 关卡编号
- 最多返回前10名
- level 参数可选，不传返回所有关卡

**接口2：获取关卡列表 [队员D 实现]**
请求：GET http://localhost:5002/api/levels

响应格式（必须按此格式解析）：
{
  "levels": [
    {"id": 1, "name": "初级", "size": 11},
    {"id": 2, "name": "中级", "size": 21},
    {"id": 3, "name": "高级", "size": 31}
  ]
}

字段说明：
- id: 关卡编号
- name: 关卡名称
- size: 迷宫大小（正方形边长，奇数）

### 功能需求：
1. 页面加载时调用 GET http://localhost:5002/api/leaderboard 获取排行榜数据
2. 前端将获取到的成绩按通关时间从小到大排序（升序，时间短的排前面）
3. 调用 GET http://localhost:5002/api/levels 获取关卡列表，页面顶部显示关卡切换按钮
4. 点击不同关卡按钮，用 ?level=N 参数重新获取该关卡排行榜
5. 用表格展示排行榜，包含：排名、玩家名、通关时间（秒）、关卡
6. 前三名用金🥇银🥈铜🥉标记
7. 每行用不同底色交替显示（斑马条纹）
8. 有一个"刷新"按钮重新获取数据
9. 顶部导航栏，可以跳转到迷宫游戏页面（../maze-game/index.html）

### 样式要求：
- 页面居中，深色背景
- 表格清晰美观，宽度 600px
- 第一名金色高亮，第二名银色高亮，第三名铜色高亮
- 斑马条纹更易阅读
- 关卡切换按钮排列在顶部

  "/* 时间复杂度：O(n log n)，使用 JavaScript 内置排序 */"

请一次性生成所有完整代码，不要省略。
